### Installation

From source code repository: 

    % cd xfce4-calculator-plugin
    % ./autogen.sh
    % make
    % make install

From release tarball:

    % tar xf xfce4-calculator-plugin-<version>.tar.bz2
    % cd xfce4-calculator-plugin-<version>
    % ./configure
    % make
    % make install
